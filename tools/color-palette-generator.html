Frequently Asked Questions

What is a color palette?

A color palette is a set of colors used in a design. A well-chosen palette creates a cohesive and visually appealing look. This tool helps you generate palettes based on a single color.

What are shades and tints?

Tints are created by adding white to a color, making it lighter. Shades are created by adding black, making it darker. This tool generates 4 tints and 4 shades for your base color.

What is a complementary color?

A complementary color is the color on the opposite side of the color wheel. It provides high contrast and can make elements stand out.

How do I copy the HEX code?

Simply click on any color swatch in the generated palette. The HEX code for that color will be copied to your clipboard.

Does this tool work offline?

Yes, this tool works entirely in your browser and does not send any data to a server.
